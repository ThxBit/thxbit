
# 📖 Service
Thxbit는 비트코인을 비롯한 다양한 암호화폐를 자동으로 매수/매도하여 수익을 창출하는 자동매매 봇입니다.  
간단한 설정만으로 알고리즘 기반의 전략을 구현하여 24시간 거래를 자동화할 수 있습니다.

## 🚀 주요 기능

- ✅ 실시간 시세 기반의 자동 매수/매도
- ✅ Bybit REST & WebSocket API 연동
- ✅ 지정가 / 시장가 주문 지원
- ✅ 기본 전략 (이동 평균선 교차, RSI 등) 내장
- ✅ 수익 로그 및 트레이딩 통계 제공
- ✅ 유연한 전략 커스터마이징
